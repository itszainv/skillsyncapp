# About SkillSync

SkillSync is a professional learning platform built with Kotlin and Jetpack Compose. 
It features a student feed with vertical scrolling lessons, interactive quizzes, and an administrative dashboard for course management.

## Features

- **Student Feed**: Vertical scrolling through educational video content.
- **Interactive Quizzes**: Knowledge checks integrated into lessons to track progress.
- **Admin Dashboard**: Manage subjects, courses, and lessons with ease.
- **Media Management**: Support for direct video uploads to Firebase Storage or manual URL entry.
- **User Progress**: Tracking of XP, levels, and learning streaks.
- **Customization**: Support for user themes and avatars.

## Setup Instructions

### Prerequisites

- Android Studio
- JDK 17
- Firebase Account

### Running the App

1. Open the project in Android Studio.
2. Let Gradle sync and download dependencies.
3. Build the project using `Build > Make Project`.
4. Run on a physical device or emulator.
